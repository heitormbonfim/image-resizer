# simple image resizer
* Dependecies:
  - Sharp

## installation
```bash
git clone <this repos link>
npm install
```
## commands
```bash
node index.mjs <file_name> <width> <quality 0-100> <new_file_name>
```
 * ex:
```bash
node index.mjs image.jpg 500 90 new-image
```
 * types supported: 
  - JPG
  - PNG
  - Webp
